# Street Eats Website (Interactive)

**Street Eats Website** is a practice project I built to improve my understanding of **HTML, CSS, and a little JavaScript**. The goal of this project was to strengthen my foundation in frontend development and learn how to create a **responsive, interactive website**.

---

## Features

- Responsive navigation bar with **dropdown menu** for mobile  
- Hover effects on links for desktop  
- Dropdown menu toggle functionality using JavaScript  
- Layout structured with **Flexbox** for content sections  
- Styled headers, buttons, and content cards  

---

## Technologies Used

- **HTML5**  
- **CSS3**  
- **JavaScript** (for dropdown menu toggle)  

---

## What I Learned

- Structuring **semantic HTML** for navigation and content  
- Implementing **responsive design** with CSS media queries  
- Adding **hover effects** and **gradients** in CSS  
- Using JavaScript to **toggle dropdown menus**  
- Writing **clean, maintainable code** for a frontend project
