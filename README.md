# Street Eats Website (Interactive Version)

**Street Eats Website (Interactive Version)** is an enhanced version of my original food truck landing page. This version focuses on adding **interactivity** to create a more engaging user experience while continuing to strengthen my frontend development skills.

---

## Overview

This project builds on the foundation of the original static landing page by introducing **JavaScript-driven features** and **UI enhancements** commonly used in modern websites. The goal was to move beyond static design and explore how interactivity improves usability and visual appeal.

---

## Features

- Responsive navigation bar with **mobile dropdown menu**
- **Hero image slider** for dynamic visual presentation
- **Scroll fade-in animations** for sections and content
- **Animated counters** (e.g., Happy Customers, Years Serving)
- Menu cards with **full-width images**

---

## Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript** (DOM manipulation, animations, Intersection Observer)

---

## What I Learned

- Enhanced a static website with **interactive JavaScript features**
- Added **scroll-based animations** using JavaScript
- Built **interactive features** like image sliders and counters
- Wrote **reusable and organized** HTML/CSS
- Applied **design consistency** with colors, overlays, and effects
